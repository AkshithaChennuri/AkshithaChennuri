1.What is HTML?

HTML stands for Hyper Text Markup Language. HTML is the standard markup language for creating Web pages. HTML describes the structure of a Web page. HTML consists of a series of elements. HTML elements tell the browser how to display the content. HTML elements label pieces of content such as "this is a heading", "this is a paragraph", "this is a link", etc.


2.What is an HTML Element?

An HTML element is defined by a start tag, some content, and an end tag: Content goes here... The HTML element is everything from the start tag to the end tag:


3. What is the difference between a tag and an element?

A tag is simply the name inside angled brackets. An element includes both opening and closing tags plus everything in between, forming a complete unit.


4.How do you add images and links?

You specify the source for an image and always include descriptive alternative text. For links, you provide the destination URL.


5.What are semantic HTML elements?

These are elements that convey meaning, like header, footer, article, section, and nav. They help with readability, accessibility, and SEO.
